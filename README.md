Analisis dan Pertanyaan
1. Mapel mana yang memiliki rata-rata nilai tertinggi?
Fisika, karena memiliki nilai rata-rata paling tinggi dibanding mapel lainnya.

2. Mapel mana yang memiliki nilai terendah?
Bahasa Indonesia, karena terdapat nilai paling rendah yaitu 60.

3. Bagaimana visualisasi membantu dalam memahami data?
Visualisasi membantu karena grafik membuat perbandingan nilai antar mapel lebih mudah dilihat dan pola datanya cepat dipahami.
